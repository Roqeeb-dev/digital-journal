# Roqeeb --- Digital Journal & Creative Studio

**Design System & Style Guide**

---

## 1. Brand Identity

**Tone:** Calm, Editorial, Personal, Premium, Intentional\
**Design Philosophy:** Minimal, Human, Thoughtful, Expressive

This system is built to feel like a _personal creative workspace_, not a
corporate product or generic blog.

---

## 2. Typography System

### Primary Display Font

Used for: - Main hero titles - Page titles - Section highlights

**Recommended:** - Playfair Display - Canela - Cormorant Garamond -
Libre Baskerville

**Fallback:** Serif system fonts

---

### Body Font

Used for: - Paragraphs - Navigation - UI text - Metadata

**Recommended:** - Inter - Geist - Manrope - Source Sans 3

---

### Type Scale

Purpose Size Weight Line Height

---

Hero Title 64--80px 500--600 1.1
Page Title 40--56px 500 1.15
Section Heading 24--32px 500 1.25
Body Text 16--18px 400 1.6
Small Meta 12--14px 400 1.4

## 4. Spacing & Layout

### Base Spacing Unit

- 4px system

### Common Spacing Scale

- 4, 8, 12, 16, 24, 32, 48, 64, 96, 128

### Layout Rules

- Wide margins
- Large vertical rhythm
- Minimal horizontal clutter
- Content should always breathe

---

## 5. Component Design Principles

### Buttons

- Text-first
- Minimal background
- Soft hover transitions
- No heavy borders

### Cards

- Soft surface
- Slight shadow
- Rounded: 12--16px
- Generous internal padding

### Inputs

- Border-light
- Soft focus rings
- Calm interaction

---

## 6. Interaction & Motion

### Allowed

- Fade in
- Gentle slide
- Soft hover lift
- Subtle blur

### Avoid

- Heavy parallax
- Bouncing motion
- Aggressive animations

---

## 7. Content Presentation

### Reading Pages

- Narrow text column
- Large margins
- Calm vertical rhythm

### Code Blocks

- Soft background
- Rounded edges
- Clear spacing

---
